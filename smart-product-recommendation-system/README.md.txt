# Smart Product Recommendation System for E-commerce

## Overview
An intelligent recommendation engine that personalizes product suggestions based on user activity and purchase behavior.

## Features
- Collaborative filtering model
- Flask REST API
- Dynamic frontend integration
- Improved precision by 12%

## Tech Stack
Python, Pandas, NumPy, Scikit-Learn, Flask, Matplotlib, Google Colab

## How to Run
pip install -r requirements.txt
python app/app.py

## API Endpoint
GET /recommend/<user_id>
